# THAIFOOD - Thai Restaurant Website

Аутентичная тайская кухня в Алматы

## Развернуто на GitHub Pages

Сайт доступен по адресу: `https://nurstar00.github.io`

### Структура проекта

- `index.html` - Главная страница
- `menu.html` - Меню ресторана
- `Onas.html` - О нас и заведениях
- `otzyv.html` - Отзывы клиентов
- `Reg.html` - Регистрация и контакты
- `style.css` - Главные стили
- `menu.css`, `Onas.css`, `otzyv.css`, `Reg.css` - Стили отдельных страниц

### Как обновить сайт

1. Внесите изменения в файлы
2. Загрузите их на GitHub
3. Изменения появятся автоматически в течение 1-2 минут
